# My blog

A Jekyll blog.

## Test locally

https://jekyllrb.com/docs/installation

```
$ brew install ruby
# add ruby and gems path
$ gem install bundler jekyll
$ bundle install
$ bundle exec jekyll server --drafts --livereload
```

Update gems:

```
$ gem update --system
$ gem update
$ gem cleanup
$ bundle update
```
